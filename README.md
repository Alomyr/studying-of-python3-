# 📚 Estudos em Python

Este repositório foi criado com o objetivo de **estudar, praticar e consolidar conhecimentos em Python**, servindo também como **portfólio de aprendizado** para futuras candidaturas a vagas de **estágio e posições júnior**.

Aqui estão reunidos exercícios, exemplos práticos e pequenos projetos que refletem minha evolução na linguagem.

---

## 🎯 Objetivos do Repositório

- Aprender os **fundamentos da linguagem Python**
- Praticar **lógica de programação**
- Desenvolver soluções claras e bem estruturadas
- Criar um histórico público de aprendizado e progresso
- Utilizar o repositório como apoio em **processos seletivos**

---

## 🧠 Conteúdos Estudados

Os estudos incluem, mas não se limitam a:

- Sintaxe básica do Python
- Tipos de dados (int, float, string, listas, tuplas, dicionários)
- Estruturas de decisão (`if`, `elif`, `else`)
- Estruturas de repetição (`for`, `while`)
- Funções
- Manipulação de strings
- List comprehensions
- Introdução à programação orientada a objetos (POO)
- Resolução de exercícios e problemas práticos

---

Python 3

Editor de código: VS Code

Sistema operacional: Linux / Windows

🚀 Como Executar os Códigos

Certifique-se de ter o Python instalado:

python --version


Clone o repositório:

git clone https://github.com/seu-usuario/nome-do-repositorio.git


Execute qualquer arquivo .py:

python nome_do_arquivo.py
