Create the Virtual Environment: Navigate to your project directory (or create one), and run:

    python3 -m venv venv

Activate the Virtual Environment:

    source venv/bin/activate

Install the Package: Now, use pip to install the package inside the isolated environment:

    pip install namelib

Deactivate (When done): When you are finished working on the project, simply run:

    deactivate
